# @fabio.caffarello/sf-tsconfig

Base TypeScript configurations used by other `@fabio.caffarello/sf-*` packages
and by projects generated by the scout-fabric. Shared TS strictness lives here.

## Files

- `base.json` — strict TS base for any project (es2022, nodenext, strict, no
  unchecked indexed access).
- `lib.json` — extends `base.json` with `composite + declaration` for buildable
  libraries.

Both configs target ES2022 and assume nodenext module resolution.

## Usage

```jsonc
// tsconfig.json
{
  "extends": "@fabio.caffarello/sf-tsconfig/base.json",
}
```

```jsonc
// tsconfig.lib.json
{
  "extends": "@fabio.caffarello/sf-tsconfig/lib.json",
  "compilerOptions": { "outDir": "dist", "rootDir": "src" },
  "include": ["src/**/*.ts"],
}
```
